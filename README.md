# Лагуна — mobile-first сайт пляжного кафе

Готовый одностраничный сайт на чистом HTML/CSS/JavaScript.

## Что внутри

- `index.html` — структура сайта.
- `styles.css` — весь визуальный стиль.
- `script.js` — переключение страниц, меню, поиск, категории, активные pills.
- `README.md` — эта инструкция.

## Как запустить

Откройте `index.html` в браузере.

## Что реализовано

- Главная страница.
- Страница меню в формате современного mobile catalogue.
- Поиск по названию, описанию, объёму и категории.
- Sticky горизонтальные категории-pills.
- Плавный скролл к разделам.
- Активное состояние категории при скролле.
- Empty state: “Нічого не знайдено”.
- Страница “Замовлення” с красивой заглушкой.
- Fixed bottom navigation panel.
- iPhone safe-area padding через `env(safe-area-inset-bottom)`.

## Как редактировать меню

Откройте `script.js` и измените массив `MENU_DATA`.
